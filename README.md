# Mini Sitio Web Educativo

## Descripción

Este proyecto consiste en un mini sitio web educativo que explica cómo funciona la Web desde que un usuario ingresa una URL hasta que el navegador muestra la página.

El sitio fue desarrollado con HTML5 y rediseñado utilizando Bootstrap 5 para mejorar la apariencia, la navegación y la experiencia del usuario.

## Framework utilizado

- Bootstrap 5

## Tecnologías utilizadas

- HTML5
- Bootstrap 5
- CSS3
