Paso 4: Renderizado HTML5

Cuando el navegador recibe la respuesta del servidor, interpreta el código HTML y construye la página web que finalmente visualiza el usuario.

Importante: El navegador interpreta el código HTML y muestra todos los elementos de la página, como texto, imágenes, tablas y formularios.

Elementos semánticos de HTML5

  • header: Encabezado de la página.
  • nav: Menú de navegación.
  • main: Contenido principal.
  • section: Agrupa contenido relacionado.
  • footer: Pie de página.

Ejemplo de código HTML

<header>
    <h1>Mi Sitio Web</h1>
</header>

<nav>
    <a href="#">Inicio</a>
</nav>

<main>
    <section>
        <p>Contenido principal</p>
    </section>
</main>

<footer>
    <p>Pie de página</p>
</footer>
