Paso 1: El usuario ingresa una URL

Cuando un usuario escribe una URL en la barra de direcciones del navegador, comienza el proceso de navegación en la Web. La URL identifica la ubicación de un recurso en Internet y permite que el navegador solicite la información al servidor correspondiente.

Ejemplo de URL:
https://www.ejemplo.com
