Paso 3: Servidor Web

Cuando el servidor recibe la solicitud HTTP enviada por el navegador, analiza la petición, localiza los recursos solicitados y prepara una respuesta para enviarla nuevamente al cliente.

Respuesta correcta: Si el recurso existe, el servidor devuelve un código HTTP 200 OK junto con la página solicitada.

Ejemplo de encabezado HTTP

HTTP/1.1 200 OK
Date: Tue, 21 May 2024 12:00:00 GMT
Server: Apache/2.4.58
Content-Type: text/html; charset=UTF-8
Content-Length: 1024
