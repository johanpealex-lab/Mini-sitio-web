Paso 2: El navegador realiza una petición HTTP

Después de que el usuario ingresa una URL, el navegador envía una solicitud HTTP al servidor para solicitar la información de la página web. Los métodos más utilizados son GET y POST.

Importante: HTTP es el protocolo que permite la comunicación entre el navegador y el servidor.

Métodos HTTP

Método Descripción Cuándo se usa
GET Solicita información al servidor. Consultar páginas o recursos.
POST Envía información al servidor. Enviar formularios o registrar datos.
